## 1. プロジェクト概要

### 1.1 目的
目黒消防団第８分団への入団を促進するための入り口となるWebサイトを構築する。

### 1.2 背景
- 現在、入団希望者や興味を持つ人が少ない
- まずは消防団の魅力を広く知ってもらい、入団のきっかけを作ることが最優先

### 1.3 ターゲット
- **主要ターゲット**: 18歳〜55歳の地域住民
- **サブターゲット**: 18歳以上の健康な方全般（管轄内在住・在勤者）

## 3. サイト構成

### 3.1 基本構造
**ハイブリッド型**: 1ページ完結型（簡略版）+ 詳細ページへの導線

- **トップページ**: 全体をざっくり把握できる簡略版
- **詳細ページ**: 各セクションの詳しい情報を独立したページで提供

### 3.2 ページ構成

#### 3.2.1 トップページ（1ページ完結型・簡略版）
以下のセクションを順番に配置：

1. **ヒーローセクション / キャッチコピー**
   - サイトのメインビジュアル
   - 分団を表現するキャッチフレーズ（※制作過程で決定）

2. **消防団の魅力・やりがい（第８分団の強み）**
   - 高い資格取得率（応急手当指導員約70%、特殊技能団員約50%、操法大会出場経験者約70%）
   - 毎月の定期訓練による指導力・技術力の向上
   - 地域防災の知識習得
   - リーダーシップ訓練
   - 充実した資格取得支援
   - 地域コミュニティへの貢献
   - 費用弁償（報酬）

3. **活動紹介（概要）**
   - 活動頻度: 基本月1〜2回（シーズン・階級により変動）
   - 主な活動内容
   - 第８分団の実力紹介（高い資格取得率、毎月の定期訓練、連続優勝実績など）
   - → 詳細ページへのリンク

4. **団員インタビュー（概要）**
   - 数名の団員の声を紹介
   - プロフィール項目: 階級、入団歴、取得資格、得意分野
   - → 詳細ページへのリンク

5. **訓練内容（概要）**
   - 防災訓練
   - 応急手当講習
   - 資機材取扱訓練
   - → 詳細ページへのリンク

6. **入団条件**
   - 18歳以上の健康な方
   - 管轄7町会が主だが、近隣でも可
   - 在勤者も可
   - 訓練に通える方

7. **よくある質問（FAQ・概要）**
   - 主要な質問3〜5件を掲載
   - → 詳細ページへのリンク

8. **SNS・活動報告への誘導**
   - Instagram、YouTubeへのリンク
   - 最新投稿のサムネイル表示（可能であれば）

9. **入団フォームへの導線**
   - CTAボタン（目立つ配置）

10. **フッター**
    - 目黒消防署の連絡先
    - プライバシーポリシーへのリンク
    - SNSリンク

#### 3.2.2 詳細ページ

1. **活動紹介詳細ページ**
   - 年間スケジュール
   - 各活動の詳細説明
   - 写真・画像ギャラリー

2. **団員インタビューページ**
   - 複数団員のインタビュー記事

4. **FAQ詳細ページ**
   - よくある質問の網羅的なリスト
   - ※内容は別途調査・ヒアリングで決定

## 5. 活動内容・アピールポイント

### 5.1 活動頻度
- 基本: 月1〜2回
- シーズン行事や階級によっては増加する場合あり

### 5.2 主なアピールポイント

#### 5.2.1 分団の実力・底力
当分団の最大の強みは、高い資格取得率に裏打ちされた指導力・活動力です。

**高い資格取得率**
- 応急手当指導員: 約70%
- 特殊技能団員: 約50%
- 操法大会出場経験者: 約70%

**継続的な訓練による技術力向上**
- 毎月定期訓練を実施
- 指導力と技術力を常に磨いている
- 先輩団員からの手厚い指導体制

**優れた実績**
- ポンプ操法大会: 平成24年度から令和7年度まで連続優勝（令和2、3、4年度はコロナ禍により大会中止）
- 応急救護普及効果確認での受賞歴

#### 5.2.2 入団者へのメリット
1. **地域防災の知識習得**
   - 災害対応のスキルが身につく

2. **組織的行動でのリーダーシップ訓練**
   - リーダーとしての能力を磨ける

3. **充実した資格取得支援**
   - 取得機会が豊富
   - 高い取得率を誇る先輩団員からの指導
   - 取得可能な資格例：
     - 上級救命講習
     - 応急手当普及員
     - 応急手当指導員
     - 可搬消防ポンプ実技講習
     - 可搬消防ポンプ等整備資格者
     - 第三級陸上特殊無線技士
     - 丙種危険物取扱者

4. **地域イベント・コミュニティへの積極的な関わり**
   - 仲間が増える
   - 地域貢献の実感

5. **応急手当の段階的習得**
   - 指導を行う団員が多数在籍（約70%が指導員資格保持）

6. **費用弁償（報酬）**
   - 基本: 4,500円
   - 災害出場時: 9,000円（実際の火災など）

## 7. 技術要件

### 7.1 ホスティング
- **GitHub Pages** を使用
- 静的サイト（HTML/CSS/JavaScript）
- 無料ホスティング
- カスタムドメイン設定可能
- 利用方法は伴走サポート必要

### 7.2 ファイル管理
- Gitベースでのアップロード
- GitHub Desktopの利用を推奨

### 7.3 レスポンシブ対応
- **モバイルファースト設計**: スマートフォンでの閲覧を優先
- スマートフォン、タブレット、PCで最適表示
- **情報の差異なし**: モバイルとPCで同じ情報を提供（表示方法のみ最適化）
- **ブレークポイント**:
  - モバイル: 〜767px
  - タブレット: 768px〜1023px
  - PC: 1024px〜

### 7.4 ブラウザ対応
- 主要ブラウザ（Chrome, Safari, Firefox, Edge）で動作確認
